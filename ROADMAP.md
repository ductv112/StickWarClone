# Stick War Clone - Roadmap

- [ ] Initialize React/Vite project with Tailwind CSS (Web-based for quick prototyping).
- [ ] Implement Basic Resource System (Gold/Crystal).

## Phase 3: Visuals & Polish
- [ ] Stickman Graphics (SVG/Canvas).
- [ ] UI for spawning units and upgrades.
- [ ] Effects and Sound.

## Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **Game Engine:** Custom Canvas/SVG logic (easier for Stickman style) or Phaser.js.
